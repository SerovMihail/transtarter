# transtarter-frontend

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
npm run serve -- --mode staging
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Run your unit tests
```
npm run test:unit
```

### Run your end-to-end tests
```
npm run test:e2e
```


### Build Web Multiply Components
```
npm run wc
```
